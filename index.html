Tutorial Deploy Website

Berikut panduan lengkap untuk mendeploy website ini dengan mudah

1

Pilih Platform Hosting

Pilih platform hosting yang mudah digunakan dan gratis untuk memulai. Rekomendasi:

  • Netlify - Mudah digunakan dengan drag & drop
  • Vercel - Baik untuk proyek frontend
  • GitHub Pages - Gratis untuk proyek open source
2

Persiapkan File Website

Simpan kode website ini dalam satu file HTML atau pisahkan menjadi file terpisah:

index.html (file utama)
style.css (file stylesheet)
script.js (file JavaScript)

Untuk kemudahan, gunakan file HTML tunggal seperti kode ini.

3

Deploy ke Netlify (Paling Mudah)

Ikuti langkah-langkah berikut:

  1. Kunjungi netlify.com dan buat akun gratis
  2. Klik "Add new site" → "Deploy manually"
  3. Drag & drop folder berisi file website Anda
  4. Tunggu proses deploy selesai (1-2 menit)
  5. Website Anda sudah online dengan URL dari Netlify
4

Deploy ke Vercel

Alternatif cepat lainnya:

  1. Kunjungi vercel.com dan buat akun
  2. Klik "Import Project" → "Deploy"
  3. Upload file ZIP atau hubungkan dengan GitHub
  4. Klik deploy dan tunggu hingga selesai
  5. Website Anda akan live dengan URL dari Vercel
5

Tambahkan Domain Kustom (Opsional)

Jika ingin menggunakan domain sendiri:

  1. Beli domain dari provider seperti Niagahoster, Domain.com, dll
  2. Di dashboard Netlify/Vercel, buka pengaturan domain
  3. Tambahkan domain kustom Anda
  4. Atur DNS records sesuai petunjuk dari platform
  5. Tunggu propagasi DNS (bisa beberapa jam)
